## Contributing

### How to Contribute
- Share configurations and rules
- Report issues and bugs
- Write tutorials and guides
- Answer community questions
- Develop plugins and extensions

### Getting Started Contributing
1. **Start small** - Answer questions in Discord/Reddit
2. **Share configs** - Post your CLAUDE.md or .cursorrules
3. **Write tutorials** - Document your workflows
4. **Build tools** - Create plugins or MCP servers
5. **Report issues** - Help improve the tools

### Contribution Guidelines
- Check existing resources before creating new ones
- Use clear, descriptive names
- Include usage examples
- Test before sharing
- Credit original sources

## Community Best Practices

### Asking Good Questions

```markdown
## Good Question Template

### What I'm trying to do
[Clear description of goal]

### What I've tried
- [Approach 1]
- [Approach 2]

### What happened
[Actual result]

### Environment
- Tool: Claude Code / Cursor / Copilot
- Version: X.Y.Z
- OS: macOS / Windows / Linux

### Code/Config (if relevant)
[Relevant snippets]
```

### Sharing Solutions

```markdown
## Solution Template

### Problem
[What wasn't working]

### Solution
[What fixed it]

### Why it works
[Brief explanation]

### Full config/code
[Complete working example]
```

### Being a Good Community Member
- Search before asking
- Provide context and examples
- Follow up with solutions
- Be respectful and patient
- Share your learnings
